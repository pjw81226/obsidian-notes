
### 연결과 포함
- **내부 링크:** `[[문서제목]]`
    
    - 클릭하면 해당 문서로 이동합니다.
        
    - `[[문서제목|보여줄이름]]` 처럼 파이프(`|`)를 쓰면 보이는 텍스트를 바꿀 수 있습니다.
        
- **문서 포함 (Embed):** `![[문서제목]]`
    
	- 느낌표(`!`)를 앞에 붙이면 링크가 아니라, **그 문서의 내용을 현재 문서에 불러와서 보여줍니다.** (이미지 넣을 때도 똑같이 씀)





### 텍스트 강조
- **굵게:** `**텍스트**` → **텍스트**
- **기울임:** `*텍스트*` → _텍스트_
- **취소선:** `~~텍스트~~` → ~~텍스트~~
- **하이라이트(형광펜):** `==텍스트==` → ==텍스트== (옵시디언 전용, 아주 유용함)

### 콜아웃

```
> [!note] 제목을 여기에 적습니다 
> 내용을 여기에 적습니다. 
> 줄바꿈도 가능합니다.
```

`[!타입]` 부분에 들어갈 수 있는 키워드는 아주 많습니다. 대소문자는 구분하지 않습니다.
- **파란색 (기본):** `[!note]`, `[!info]`
- **초록색 (팁, 성공):** `[!tip]`, `[!success]`, `[!check]`
- **노란색 (질문, 경고):** `[!question]`, `[!warning]`, `[!caution]`
- **빨간색 (위험, 에러):** `[!failure]`, `[!danger]`, `[!bug]`
- **보라색 (인용, 예시):** `[!quote]`, `[!example]`


### 코드입력
- **인라인 코드 (문장 중간):** 백틱(`) 하나로 감쌉니다.
    - 작성: `String` 클래스는 불변이다.
    - 결과: `String` 클래스는 불변이다.
        
- **코드 블록 (여러 줄):** 백틱 3개(```) 뒤에 언어를 적습니다.
```java
public class Main { public static void main(String[] args) {
		System.out.println("Hello"); 
	} 
}
```

### 리스트와 할일
**순서 없는 목록:** `-` 또는 `*` 뒤에 띄어쓰기.

**순서 있는 목록:** `1.` 뒤에 띄어쓰기.

**체크박스:** `- [ ]` (대괄호 안에 공백)
    
    - `- [ ] JVM 구조 공부하기`
        
    - `- [x] GC 알고리즘 정리 완료` (x를 넣으면 체크됨)
