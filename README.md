# Jangwoo Obsidian Notes
개인 학습, 실험, 정리 노트를 Obsidian으로 기록하고 GitHub로 버전 관리합니다.
이 저장소는 제 공부 흐름과 지식 베이스를 공개, 공유하기 위한 공간입니다.

## 무엇이 들어있나요
- 주제별 노트, 개념 정리, 시행착오 기록
- 예제 코드, 링크 모음, 체크리스트
- 필요 시 이미지, 첨부자료(가능한 한 가벼운 범위)

## 사용 방법
1. 이 저장소를 clone 합니다
2. Obsidian에서 `Open folder as vault`로 열면 그대로 볼 수 있습니다
